# NGS基礎実習（森岡担当）
NGS解析の練習として、ChIP-seqを中心としたの実習を行います。RNA-seqやChIP-seqは、それぞれの異なるサーバで解析していただきますので、コースよりお選びください。2,3番については、**自分で読んで、調べて解析していただくコースになりますので、当日のレクチャーがありません。2,3の質問は講義の最後にどうぞ。**
